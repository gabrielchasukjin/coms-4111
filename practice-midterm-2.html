CSEE 4121 — Practice Midterm 2

Computer Systems for Data Science | Spring 2026 | Harder Difficulty

  • Time: 90 minutes
  • Questions: 20 — mix of MCQ, short answer, SQL, and trace problems
  • Total Points: 100
  • Difficulty: Slightly above midterm level
1 Performance 5 pts

A program spends 80% of its time in a parallelizable section. You have unlimited cores. What is the maximum possible speedup according to Amdahl's Law?

2 WAL 5 pts

A system writes data pages to disk before writing the corresponding log records. After a crash, which ACID property is violated and why?

3 Isolation 5 pts

PostgreSQL's default isolation level is Read Committed. Which of the following anomalies can still occur at this level? Select the most complete answer.

4 SQL / Subqueries 5 pts
Product(PName, Price, Category, Manufacturer) Company(CName, Country, NoOfEmployees)

Write a SQL query to find all products that are more expensive than every other product in the same category (i.e., the most expensive product per category). Do NOT use GROUP BY.

5 SQL / Aggregation 5 pts

A table employees(id, name, dept) has 100 rows. The dept column is NULL for 15 employees. What does SELECT COUNT(*), COUNT(dept) FROM employees return?

6 SQL / Joins 5 pts

Table A has 10 rows. Table B has 8 rows. No rows in A match any row in B on the join condition. How many rows does A FULL OUTER JOIN B return?

7 Relational Model 5 pts

A Students table has a primary key id. An Enrollments table has a foreign key student_id referencing it with ON DELETE CASCADE. If you delete a student, what happens to their enrollments?

8 Protocol Buffers 5 pts

Why should you avoid using required fields in Protocol Buffers and prefer optional?

9 Locking 5 pts

Under Strict 2PL, when are locks released?

10 Deadlocks 5 pts

Three transactions under Strict 2PL:

T1: holds X(A), requests S(B) T2: holds X(B), requests S(C) T3: holds X(C), requests S(A)

Draw the waits-for graph. Is this a deadlock? If so, what is the cycle?

11 Serializability 5 pts

Given this schedule with three transactions:

w1(A), r2(A), w2(B), r3(B), w3(A)

List all conflicts, draw the conflict graph edges, and determine if this is conflict serializable. If yes, give an equivalent serial order.

12 Bloom Filters 5 pts

You have a Bloom filter with m = 10 bits and k = 2 hash functions. After inserting elements X and Y, the bit array is: [0, 1, 0, 1, 0, 0, 1, 0, 1, 0]. You query element Z and get h₁(Z) = 1, h₂(Z) = 6. What does the Bloom filter report?

13 Bloom Filters 5 pts

Why can't you delete elements from a standard Bloom filter by setting bits back to 0?

14 Indexing 5 pts

A table is physically sorted by id (primary key). You create a secondary index on salary. Must this secondary index be dense or sparse, and why?

15 Caching 5 pts

Cache size = 3. Sequence: A, B, C, A, B, D, A, B, C, D. Complete the FIFO trace. How many misses occur?

16 Caching 5 pts

Cache size = 3. Cache currently holds {A, B, C}. Next request is D. Remaining future sequence after D is: A, C, B, A. Under OPT (Belady's), which item is evicted?

17 Networking 5 pts

A video call application uses UDP instead of TCP. Which of the following best explains why?

18 Networking 5 pts

When your laptop connects to a new Wi-Fi network, it doesn't have an IP address yet. Which protocol assigns it one, and at which layer does this operate?

19 Distributed Systems 5 pts

In Two-Phase Commit, the coordinator sends PREPARE and all participants vote YES. The coordinator writes COMMIT to its log and starts sending COMMIT messages. It crashes after sending COMMIT to participant 1 but before sending it to participant 2. What happens when the coordinator recovers?

20 Partitioning 5 pts

You have 4 nodes using hash partitioning with node = hash(key) mod 4. You add a 5th node, changing the formula to mod 5. Approximately what fraction of keys must be moved?
